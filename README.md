# 🎂 Fluffy Bakes Dashboard

An interactive Streamlit dashboard for bakery business owners to explore sales trends, customer loyalty, peak performance days, and machine learning insights to drive smart business decisions.

## 📊 Overview

**Fluffy Bakes Dashboard** is designed to help bakery shop owners:

- Understand top-selling items
- Track revenue trends over months
- Discover peak sales days
- Analyze payment preferences
- Visualize new vs repeat customers
- Predict customer loyalty using machine learning
- Receive data-driven growth suggestions

---

## 🚀 Features

✅ **Interactive Visualizations**  
✅ **Customer Behavior Classification (ML)**  
✅ **Business Strategy Suggestions**  
✅ **Responsive Layout with Modern Styling**  
✅ **Data-Driven Decision Making for SMEs**

---

## 🧠 Machine Learning Insight

Uses `RandomForestClassifier` to classify customers as repeat or new based on:

- Total Spend
- Order Frequency
- Unique Items Ordered

Includes:
- Classification Report
- Confusion Matrix
